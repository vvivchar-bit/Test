# Тестування вимог програмної системи «Готель»

## Опис

У репозиторії розміщено матеріали для роботи з тестування вимог навчального проєкту **«Програмна система Готель»**.

У межах роботи було виконано:

- створення вимог у ReQtest;
- створення тест-кейсів для перевірки вимог;
- запуск Test Run у ReQtest;
- фіксацію результатів тестування;
- побудову матриці трасування вимог;
- перевірку браузерного прототипу системи «Готель».

## Основний результат

У тестовому запуску **TR-01 System-Hotel requirements testing** було виконано 6 тест-кейсів. Усі тест-кейси отримали результат **OK / Passed**.

Робота демонструє практичне тестування вимог проєкту «Готель»: вимоги були внесені до ReQtest, для них створено тест-кейси, виконано тестовий запуск і сформовано матрицю трасування. Це підтверджує зв’язок між вимогами, перевірками та результатами тестування.
